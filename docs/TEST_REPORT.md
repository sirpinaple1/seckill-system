## 4. 问题与优化

### 4.1 发现的问题
1. 高并发下部分请求响应较慢
2. 库存扣减存在并发竞争
3. 前端实时数据更新延迟

### 4.2 优化建议
1. 增加 Redis 集群提高并发能力
2. 优化分布式锁实现
3. 使用 WebSocket 推送实时数据
4. 添加 Sentinel 限流保护
