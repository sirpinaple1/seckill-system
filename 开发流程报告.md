# 秒杀系统开发流程报告

## 一、项目背景

### 1.1 问题描述
公司原有单体电商平台，每逢促销活动（如双11、限时抢购）期间，系统频繁崩溃，无法支撑突发的海量并发请求，客户投诉严重。

### 1.2 项目目标
作为后端核心开发，负责设计并实现一套高可用秒杀系统，要求支持10万+ QPS，确保活动期间系统稳定运行。

---

## 二、技术架构

### 2.1 整体架构图
```
用户请求 → Nginx负载均衡 → 秒杀服务 → Redis分布式锁
                                              ↓
                                    RabbitMQ消息队列
                                              ↓
                                    订单服务 → MySQL持久化
```

### 2.2 技术选型

| 技术栈 | 版本 | 用途 |
|--------|------|------|
| Spring Boot | 3.2.0 | 后端框架 |
| Vue | 2.7 | 前端框架 |
| Redis | 7.0 | 分布式锁、缓存 |
| RabbitMQ | 3.12 | 消息队列 |
| MySQL | 8.0 | 持久化存储 |
| Element UI | 2.15 | 前端组件库 |

---

## 三、核心功能模块

### 3.1 后端模块

| 模块 | 说明 |
|------|------|
| `SeckillController` | 秒杀API接口 |
| `SeckillService` | 秒杀业务逻辑 |
| `RedisLock` | Redis分布式锁 |
| `OrderProducer` | 订单消息生产者 |
| `OrderConsumer` | 订单消息消费者 |

### 3.2 核心流程

1. **秒杀请求** → 分布式锁校验 → 库存原子扣减 → 发送消息队列
2. **订单处理** → 异步消费 → 创建订单 → 返回结果

## 四、开发步骤

### 4.1 环境准备
```bash
# 安装 Java 17
sudo apt install openjdk-17-jdk

# 安装 Redis
sudo apt install redis-server

# 安装 RabbitMQ
sudo apt install rabbitmq-server

# 安装 MySQL
sudo apt install mysql-server

# 安装 Maven
sudo apt install maven
```

### 4.2 数据库设计
```sql
-- 秒杀商品表
CREATE TABLE seckill_product (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    product_name VARCHAR(200),
    original_price DECIMAL(10,2),
    seckill_price DECIMAL(10,2),
    stock INT,
    start_time DATETIME,
    end_time DATETIME
);

-- 秒杀订单表
CREATE TABLE seckill_order (
    order_no VARCHAR(64) PRIMARY KEY,
    product_id BIGINT,
    user_id BIGINT,
    price DECIMAL(10,2),
    status VARCHAR(20)
);
```

### 4.3 后端启动
```bash
cd backend
mvn clean package
java -jar target/seckill-system-1.0.0.jar
```

### 4.4 前端启动
```bash
cd frontend
npm install
npm run serve
```

## 五、核心代码说明

### 5.1 Redis分布式锁
```java
// 获取锁
boolean lock = redisLock.tryLock(lockKey, userId, 10);

// 扣减库存
Long stock = redisTemplate.opsForValue().decrement(stockKey);

// 释放锁
redisLock.unlock(lockKey, userId);
```

### 5.2 消息队列
```java
// 发送订单消息
orderProducer.sendOrder(message);

// 消费订单消息
@RabbitListener(queues = "seckill.order.queue")
public void handleOrder(OrderMessage message) {
    orderService.createOrder(message);
}
```
